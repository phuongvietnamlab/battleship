# Phase 18 — Plan 03: Queue timeout integration + Client-side adjustments

## Objective

Wire the 15-second queue timeout into the existing matchmaking sweep so players waiting too long are automatically paired with a bot. Adjust client-side to remove the manual 30s bot offer (now replaced by auto-match). Configure the timeout via environment variable.

## Requirements Addressed

- BOT-QM-03: After 15s in queue without human match → auto-pair with bot
- BOT-QM-04: Bot selection — random available bot; if all busy, retry every 5s
- BOT-QM-09: Real coin wagering — bot matches player's stake
- BOT-QM-12: Auto-replenish bot wallet before match
- BOT-QM-13: Player should NOT know they're playing a bot
- BOT-QM-15: Configurable timeout via BOT_MATCH_TIMEOUT_MS (default 15000)
- BOT-QM-16: No new user-facing strings needed

### Task 2: Bot queue sweep logic

Add `tryBotMatch()` function to server.js, called from the existing sweep interval:

```javascript
/**
 * Check if any queue entry has been waiting longer than BOT_MATCH_TIMEOUT_MS.
 * If so, and no human pair is available, match them with a random bot.
 */
async function tryBotMatch() {
  const now = Date.now();

  for (const type of ['free', 'wagered']) {
    const q = queues[type];
    for (const [queueKey, entry] of [...q.entries()]) {
      // Skip entries currently being paired
      if (entry.pairing) continue;
      // Skip entries that haven't waited long enough
      if (now - entry.enqueuedAt < BOT_MATCH_TIMEOUT_MS) continue;
      // Skip if socket is dead
      if (!socketIsLive(entry.socketId)) continue;

      // Check if a human pair is possible right now
      const otherEntries = [...q.values()].filter(e => e !== entry && !e.pairing);
      const humanPairAvailable = type === 'wagered'
        ? otherEntries.some(e => e.stake === entry.stake)
        : otherEntries.length > 0;

      // If a human pair exists, let normal tryPair handle it
      if (humanPairAvailable) continue;

      // Find an available bot
      const busyBotIds = new Set(activeBotMatches.keys());
      const availableBots = await getAvailableBots(busyBotIds);

      if (availableBots.length === 0) {
        // All bots busy — will retry on next sweep (every 5s per BOT-QM-04)
        console.log(`[bot-match] all bots busy, retrying in ${QUEUE_SWEEP_MS}ms for ${entry.clientId}`);
        continue;
      }

      // Pick a random bot
      const bot = availableBots[Math.floor(Math.random() * availableBots.length)];

      // Replenish bot wallet if needed (BOT-QM-12)
      await replenishBotWallet(bot.id);

      // Check if bot can afford the stake (for wagered matches)
      if (type === 'wagered' && entry.stake > 0) {
        const botBalance = await getWalletBalance(bot.id);
        if (botBalance < entry.stake) {
          // This bot can't afford it — try another
          const affordableBot = availableBots.find(async b => {
            const bal = await getWalletBalance(b.id);
            return bal >= entry.stake;
          });
          if (!affordableBot) {
            // No bot can afford → will be free match (handled in createBotMatchRoom)
            console.log(`[bot-match] no bot can afford stake ${entry.stake}, creating free bot match`);
          }
        }
      }

      // Remove entry from queue (same synchronous guard as tryPair)
      entry.pairing = true;
      q.delete(queueKey);

      // Clear socket queue state
      const sock = io.of('/').sockets.get(entry.socketId);
      if (sock) {
        sock.data.queueType = null;
        sock.data.queueClientId = null;
      }

      // Create bot match
      try {
        await createBotMatchRoom(entry, bot, type);
        console.log(`[bot-match] auto-matched ${entry.clientId} with bot ${bot.display_name} after ${now - entry.enqueuedAt}ms`);
      } catch (err) {
        console.error('[bot-match] createBotMatchRoom failed:', err.message);
        // Re-queue human if match creation fails
        entry.pairing = false;
        q.set(queueKey, entry);
        if (sock) {
          sock.data.queueType = type;
          sock.data.queueClientId = entry.clientId;
        }
      }

      // Only match one entry per sweep cycle to avoid thundering herd
      break;
    }
  }
}
```

- Bot wins by forfeit
- Call `endBotMatch(code, botUserId, humanUserId, 'disconnect')`
- Clean up room and bot tracking

// In the existing disconnect handler, add:

}
