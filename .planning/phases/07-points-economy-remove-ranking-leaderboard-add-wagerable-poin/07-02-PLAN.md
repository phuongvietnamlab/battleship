### Task 4: Remove ranked/leaderboard code from `server.js`

**Type:** code
**File(s):** `server.js` (modify)

**Actions:**
1. **Remove ranked queue constants** (lines ~200-216):
   - `RANKED_WINDOW_START`, `RANKED_WINDOW_STEP`, `RANKED_WINDOW_CAP`

4. **Remove `GET /api/leaderboard` route** (lines ~242-258): Delete the entire `app.get("/api/leaderboard", ...)` handler.

5. **Remove `rankedWindow()` function** (lines ~1144-1155): Delete entirely.

6. **Replace `findPair()` function** — remove the ranked branch. Replace with:
   ```javascript
   function findPair(type, entries) {
     if (entries.length < 2) return null;
     if (type === "wagered") {
       // Match only entries with the same stake level
       const byStake = {};
       for (const e of entries) {
         (byStake[e.stake] = byStake[e.stake] || []).push(e);
       }
       for (const group of Object.values(byStake)) {
         if (group.length >= 2) return [group[0], group[1]];
       }
       return null;
     }
     return [entries[0], entries[1]];
   }
   ```

7. **Remove `emitQueueStatus()`** (lines ~1220-1235): Delete the entire function that sends ranked window/waitSec updates.

8. **Update `tryPairAll()`** — change from:
   ```javascript
   function tryPairAll() {
     tryPair("casual");
     tryPair("ranked");
     emitQueueStatus();
   }
   ```
   to:
   ```javascript
   function tryPairAll() {
     tryPair("free");
     tryPair("wagered");
   }
   ```

9. **Update `removeFromQueues()`** — change the iteration from `["casual", "ranked"]` to `["free", "wagered"]`.

10. **Update `createRoom` handler** (line ~1319-1340):
    - Remove the ranked guards: `if (ranked && mode === "advance") return cb(...)` and `if (ranked && socket.data.userId == null) return cb(...)`
    - Remove `ranked` variable declaration from arg parsing
    - Remove `ranked` from the rooms[code] object literal (or set it to `false` / remove entirely)

11. **Update `joinRoom` handler** — remove the ranked guard: `if (room.ranked && socket.data.userId == null) return cb(...)` (line ~1470)

12. **Update `joinQueue` handler** (lines ~1375-1410):
    - Change type normalization from `=== "ranked" ? "ranked" : "casual"` to `=== "wagered" ? "wagered" : "free"`
    - Remove the ranked account guard (`RANKED_REQUIRES_ACCOUNT`)
    - Remove the `getPlayerRating()` call and `rating`/`rd` variables (handled differently in Plan 03)

13. **Update all `recordMatch` calls** — remove the trailing `room.ranked` argument from every call site:
    - `scheduleSeatRelease` disconnect handler (~line 668)
    - `endGameForfeit` (~line 1007)
    - `doShot` win path (~line 1058)
    - `leaveRoom` handler (~line 1786)

14. **Remove `ranked` from room serialization/restoration** — in `serializeRooms()` and `restoreRooms()`, remove `ranked: !!r.ranked` / `ranked: !!s.ranked`.

15. **Remove `ranked` from `createMatchedRoom`** — remove `const ranked = type === "ranked";` and `ranked` from the room object. Remove `ranked` from the `matchFound` emit payload.

16. **Remove leaderboard-related TEST_EXPORTS**:
    - Remove `leaderboardLimiter`
    - Remove `getLbCache: () => lbCache`
    - Remove `resetLbCache: () => { ... }`
    - Remove `rankedWindow`
    - Update `queues` reference (already renamed in step 2)

**Acceptance criteria:**
- [ ] No reference to "ranked", "leaderboard", "rankedWindow", or "lbCache" in server.js
