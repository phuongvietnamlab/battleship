# Plan 15-03: Power-up Implementations — Server-side Game Logic

## Objective
Implement the 4 new power-up abilities on the server: Sonar Ping, Cross Missile, Scatter Blast, and Decoy hit behavior. All power-ups always consume the turn.

## Tasks

### Task 1: Add `forceEndTurn` mechanism to doShot

**Files:** `server.js`

The current `doShot` keeps turn on hit (`let keep = anyHit`). Power-ups need to always end turn.

1. Add optional parameter: `doShot(room, clientId, cells, { forceEndTurn = false } = {})`
2. After hit detection, if `forceEndTurn` is true:
   ```javascript
   if (forceEndTurn) keep = false;
   ```
3. This ensures Cross Missile and Scatter Blast always pass the turn even on hits
4. Normal `fire` event continues to pass `forceEndTurn: false` (default) — keeps "hit = keep turn" behavior

### Task 2: Implement Sonar Ping handler

**Files:** `server.js`

In the `useAbility` handler, add `type === "sonar"`:

```javascript
if (type === "sonar") {
  const { axis, index } = arg; // axis: "row"|"col", index: 0-10
  if (!["row", "col"].includes(axis)) return cb({ ok: false, code: "BAD_AXIS" });
  if (!Number.isInteger(index) || index < 0 || index >= BOARD) return cb({ ok: false, code: "BAD_INDEX" });
  
  me.inv.sonar--;
  me.timeouts = 0;
  
  const opp = opponentOf(room, clientId);
  const oppData = room.players[opp];
  let found = false;
  if (oppData && oppData.occ) {
    for (const k of oppData.occ) {
      const [r, c] = k.split(",").map(Number);
      if (axis === "row" && r === index) { found = true; break; }
      if (axis === "col" && c === index) { found = true; break; }
    }
  }
  
  // Consume turn
  giveTurn(room, opp, clientId);
  for (const id of room.order) emitToClient(room, id, "turnUpdate", { yourTurn: room.turn === id });
  armTurnTimer(room);
  emitInv(room, clientId);
  
  return cb({ ok: true, type: "sonar", result: found ? "YES" : "NO", axis, index });
}
```

Key points:
- Scans `occ` which includes decoy cell (intentional — decoy fools sonar)
- Always consumes turn
- Only responds to the activating player (secrecy: opponent doesn't know what was scanned)

### Task 3: Implement Cross Missile handler

**Files:** `server.js`

Move from `fire` event to `useAbility`:

```javascript
if (type === "cross") {
  if (!inBounds(r, c)) return cb({ ok: false, code: "BAD_CELL" });
  if ((me.inv.cross || 0) <= 0) return cb({ ok: false, code: "NO_POWERUP" });
  
  me.inv.cross--;
  me.timeouts = 0;
  
  room.resolving = true;
  let summary;
  try {
    summary = doShot(room, clientId, expandCells("cross", r, c), { forceEndTurn: true });
  } finally {
    room.resolving = false;
  }
  
  emitInv(room, clientId);
  return cb(Object.assign({ type: "cross" }, summary));
}
```

Keep existing `expandCells("cross", r, c)` function (returns center + 4 adj within bounds).

### Task 4: Implement Scatter Blast handler

**Files:** `server.js`

```javascript
if (type === "scatter") {
  const cand = [];
  for (let rr = 0; rr < BOARD; rr++) for (let cc = 0; cc < BOARD; cc++) {
    const k = rr + "," + cc;
    if (!me.hits.has(k)) cand.push([rr, cc]);
  }
  if (!cand.length) return cb({ ok: false, code: "NO_CELLS" });
  if ((me.inv.scatter || 0) <= 0) return cb({ ok: false, code: "NO_POWERUP" });
  if (room.resolving) return cb({ ok: false, code: "BAD_STATE" });
  
  me.inv.scatter--;
  me.timeouts = 0;
  
  const n = Math.min(cand.length, 2 + Math.floor(Math.random() * 2)); // 2-3 cells
  const pick = [];
  for (let i = 0; i < n; i++) pick.push(cand.splice(Math.floor(Math.random() * cand.length), 1)[0]);
  
  room.resolving = true;
  let summary;
  try {
    summary = doShot(room, clientId, pick, { forceEndTurn: true });
  } finally {
    room.resolving = false;
  }
  
  emitInv(room, clientId);
  return cb(Object.assign({ type: "scatter" }, summary));
}
```

### Task 5: Verify Decoy hit behavior in doShot

**Files:** `server.js`

No code changes needed for Decoy — it works automatically:
1. Decoy cell is in `occ` → `hit = oppData.occ.has(k)` returns `true` → reports HIT
2. Decoy cell is NOT in `ships` array → `sunkShipCount` never counts it as sunk
3. Win condition: `sunkCount >= FLEET.length` — only real ships count

Verify edge cases:
- Cross Missile hits decoy: reports HIT, no sunk notification → correct
- Scatter Blast hits decoy: reports HIT, no sunk notification → correct
- Sonar scans decoy row/col: returns YES → correct (decoy fools sonar)
- All 5 real ships sunk but decoy not hit: game ends → correct
