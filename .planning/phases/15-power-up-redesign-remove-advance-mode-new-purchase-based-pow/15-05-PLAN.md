# Plan 15-05: Client UI — Battle Phase Power-ups

## Objective
Build the battle-phase UI for using purchased power-ups: Sonar Ping (drag-and-drop), Cross Missile (aim mode), Scatter Blast (one-click), and update the PowerBar to show only purchased items.

## Tasks

### Task 1: Rewrite PowerBar for new system

**Files:** `public/app.jsx`

Replace old PowerBar with new version that only shows purchased power-ups:

```jsx
function PowerBar({ inv, aim, onPower, myTurn }) {
  const items = [
    { type: "sonar", icon: "🔊", name: t("pw.sonar") },
    { type: "cross", icon: "➕", name: t("pw.cross") },
    { type: "decoy", icon: "🪤", name: t("pw.decoy") },  // decoy won't appear (used at placement)
    { type: "scatter", icon: "🌠", name: t("pw.scatter") },
  ].filter(({ type }) => (inv[type] || 0) > 0);
  
  if (items.length === 0) return null; // no power-ups purchased
  
  return (
    <div className="power-bar">
      {items.map(({ type, icon, name }) => (
        <button key={type} disabled={!myTurn}
          className={"power-btn" + (aim === type ? " aiming" : "")}
          onClick={() => onPower(type)}>
          <span className="pi">{icon}</span>
          <span className="pn">{name}</span>
          <span className="pc">×{inv[type]}</span>
        </button>
      ))}
    </div>
  );
}
```

Key change: PowerBar always renders (no mode check), but returns null if no power-ups purchased.

### Task 2: Implement Sonar Ping UX — Drag-and-drop

**Files:** `public/app.jsx`

When player activates Sonar:
1. `setAim("sonar")` → enters sonar mode
2. Show overlay with 2 draggable "lego blocks":
   - Horizontal bar (≡) → represents row scan
   - Vertical bar (|||) → represents column scan
3. Drag interaction (reuse placement DnD pattern):
   - `onPointerDown` on block → start drag
   - `onPointerMove` → move ghost element following pointer
   - `onPointerUp` on enemy grid → calculate which row/col was targeted
4. On successful drop:
   - Determine axis ("row" or "col") based on which block was dragged
   - Determine index from grid position (row 0-10 or col 0-10)
   - Emit `socket.emit("useAbility", { type: "sonar", axis, index }, cb)`
   - Show result in log: "🔊 Sonar scanned row B — YES!" or "🔊 Sonar scanned column 5 — NO"
   - Clear aim state
5. Cancel: tap the Sonar button again or press outside grid

### Task 5: Update activatePower function

**Files:** `public/app.jsx`

Rewrite `activatePower(type)`:
```javascript
function activatePower(type) {
  if (!myTurn || (inv[type] || 0) <= 0) return;
  
  if (type === "sonar") {
    setAim(a => a === "sonar" ? null : "sonar");
    return;
  }
  if (type === "cross") {
    setAim(a => a === "cross" ? null : "cross");
    return;
  }
  if (type === "scatter") {
    Sound.fire();
    socket.emit("useAbility", { type: "scatter" }, (res) => {
      if (!res.ok) { addLog(errText(res)); return; }
      addLog(t("log.scatterBoom"));
      applyShotResult(res, t("pw.scatter"), true);
    });
    return;
  }
}
```
