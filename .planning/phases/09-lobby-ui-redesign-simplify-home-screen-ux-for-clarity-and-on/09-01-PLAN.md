# Phase 09 — Plan 01: Card-based lobby layout + BottomSheet component

## Objective

Replace the current flat list of 10+ buttons with a clean card-based layout: 1 hero CTA + 2 secondary cards + a BottomSheet for progressive disclosure. The entire lobby must fit in one mobile viewport without scrolling.

## Requirements Addressed

- LOBBY-01: Single primary CTA "Chơi nhanh / Quick Play"
- LOBBY-02: Two compact secondary cards side-by-side (Bot + Friends)
- LOBBY-03: Bot card → bottom sheet with difficulty picker
- LOBBY-04: Friends card → bottom sheet with Create/Join room
- LOBBY-07: Fits one mobile viewport (≤ 667px height) without scrolling
- LOBBY-08: Progressive disclosure via bottom sheets
- LOBBY-11: Responsive stacking on very small screens

## Tasks

### Task 2: Restructure Lobby into card-based layout

**File:** `public/app.jsx` — replace the Lobby `return(...)` JSX

New structure:
```jsx
<div className="lobby">
  <h2>{t("lobby.title")}</h2>
  {error && <div className="error">...</div>}

  {/* Hero CTA — Quick Play */}
  <button className="btn primary hero-cta" onClick={onQuickMatch}>
    <span className="hero-icon">⚡</span>
    <span className="hero-text">
      <strong>{t("lobby.quickPlay")}</strong>
      <small>{t("lobby.quickPlaySub")}</small>
    </span>
  </button>

  {/* Secondary cards row */}
  <div className="lobby-cards">
    <button className="lobby-card" onClick={() => setBotSheetOpen(true)}>
      <span className="card-icon">🤖</span>
      <strong>{t("lobby.botCard")}</strong>
      <small>{t("lobby.botCardSub")}</small>
    </button>
    <button className="lobby-card" onClick={() => setFriendSheetOpen(true)}>
      <span className="card-icon">👥</span>
      <strong>{t("lobby.friendCard")}</strong>
      <small>{t("lobby.friendCardSub")}</small>
    </button>
  </div>

  {/* Mode toggle — compact, inline */}
  <div className="mode-toggle-compact">
    <button className={mode === "classic" ? "active" : ""} onClick={() => setMode("classic")}>
      {t("mode.classic")}
    </button>
    <button className={mode === "advance" ? "active" : ""} onClick={() => setMode("advance")}>
      {t("mode.advance")}
    </button>
  </div>

  {/* Wager section — only for logged-in users */}
  {authUser && balance !== null && (
    <div className="wager-strip">
      <span className="wager-balance">🪙 {balance}</span>
      <div className="wager-chips">
        {[10,25,50,100].map(s => (
          <button key={s} className={selectedStake === s ? "chip active" : "chip"}
            onClick={() => setSelectedStake(s)} disabled={balance < s}>{s}</button>
        ))}
      </div>
      <button className="btn steel compact" onClick={() => onWageredMatch(selectedStake)}
        disabled={balance < selectedStake}>
        {t("queue.wageredMatch")}
      </button>
    </div>
  )}

  {/* Footer utilities */}
  <div className="lobby-footer">
    <button className="btn ghost compact" onClick={onHelp}>{t("help.open")}</button>
    {!authUser && hasPlatformAuth && (
      <PasskeyButton clientId={clientId} onAuthSuccess={onEmailAuthSuccess} mode="login" />
    )}
  </div>

  {/* Bottom Sheets */}
  <BottomSheet open={botSheetOpen} onClose={() => setBotSheetOpen(false)} title={t("bot.selectTier")}>
    {/* Bot difficulty cards with descriptions */}
    <div className="sheet-options">
      {VALID_TIERS.map(tier => (
        <button key={tier} className="sheet-option" onClick={() => { saveBotTier(tier); setSelectedTier(tier); onBot(tier); setBotSheetOpen(false); }}>
          <strong>{t("bot." + tier)}</strong>
          <small>{t("bot." + tier + "Desc")}</small>
        </button>
      ))}
    </div>
  </BottomSheet>

  <BottomSheet open={friendSheetOpen} onClose={() => setFriendSheetOpen(false)} title={t("lobby.friendTitle")}>
    <button className="btn steel" onClick={() => { onCreate(mode, roomStake); setFriendSheetOpen(false); }}>
      {t("lobby.createRoom")}
    </button>
    <div className="divider">{t("common.or")}</div>
    <div className="field">
      <label>{t("lobby.enterCodeLabel")}</label>
      <input className="code-input" maxLength={5} placeholder="ABCDE"
        value={code} onChange={e => setCode(e.target.value.toUpperCase())}
        onKeyDown={e => e.key === "Enter" && code && onJoin(code)} />
    </div>
    <button className="btn steel" disabled={code.length < 4}
      onClick={() => { onJoin(code); setFriendSheetOpen(false); }}>
      {t("lobby.joinBtn")}
    </button>
  </BottomSheet>
</div>
```

**New state variables:**
```jsx
const [botSheetOpen, setBotSheetOpen] = useState(false);
const [friendSheetOpen, setFriendSheetOpen] = useState(false);
```

**Acceptance:**
- Lobby renders as: Hero button → 2 cards → mode toggle → (wager strip if logged in) → footer
- Bot card opens BottomSheet with 4 difficulty options
- Friends card opens BottomSheet with Create/Join room
- All fits on one mobile screen without scroll
- Existing functionality preserved (onBot, onQuickMatch, onCreate, onJoin all work)
