### Task 2: First-time onboarding pulse

**File:** `public/app.jsx` + `public/style.css`

Add a subtle pulse animation + tooltip on the hero CTA for first-time visitors:

```jsx
const [showOnboarding, setShowOnboarding] = useState(() => {
  try { return !localStorage.getItem("lobby_onboarded"); } catch { return true; }
});

// After first interaction with any lobby action:
function dismissOnboarding() {
  setShowOnboarding(false);
  try { localStorage.setItem("lobby_onboarded", "1"); } catch {}
}
```

On the hero CTA:
```jsx
<button className={"btn primary hero-cta" + (showOnboarding ? " onboarding-pulse" : "")}
  onClick={() => { dismissOnboarding(); onQuickMatch(); }}>
  ...
</button>
{showOnboarding && (
  <div className="onboarding-hint">{t("lobby.onboardingHint")}</div>
)}
```

**CSS:**
```css
@keyframes onboarding-pulse {
  0%, 100% { box-shadow: 0 8px 22px rgba(240,193,75,.28); }
  50% { box-shadow: 0 8px 32px rgba(240,193,75,.55), 0 0 0 6px rgba(240,193,75,.15); }
}
.hero-cta.onboarding-pulse { animation: onboarding-pulse 2s ease-in-out infinite; }
.onboarding-hint {
  text-align: center; font-size: 12px; color: var(--gold); margin: -8px 0 12px;
  animation: fadeIn .5s ease .3s both;
}
@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } }
```

**i18n:**
```js
// en
"lobby.onboardingHint": "Tap here to start playing!",
// vi
"lobby.onboardingHint": "Bấm vào đây để bắt đầu chơi!",
```

**Acceptance:**
- First visit: hero button pulses gently, hint text below
- After any lobby action: pulse and hint disappear permanently
- Returning users never see the pulse again
- Not intrusive — just a gentle visual guide

### Task 4: Accessibility pass

**Files:** `public/app.jsx`, `public/style.css`

- Add `aria-label` to lobby cards (e.g., `aria-label={t("lobby.botCard") + " - " + t("lobby.botCardSub")}`)

- Escape key closes BottomSheet
- Hero CTA: `aria-describedby` for onboarding hint

**Acceptance:**
- VoiceOver/TalkBack can navigate lobby cards and sheets
- Escape closes any open sheet
- Tab key stays trapped inside open sheet
- Screen reader announces card roles and states
