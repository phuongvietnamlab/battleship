Slice 5 of the ranked vertical: an admin can roll the season. `scripts/season-reset.js` is a standalone Node CLI (run on the server box only — D-13, zero public HTTP surface) that, in a single transaction: inserts a new `seasons` row (UNIQUE label = idempotency guard), archives every current `ratings` row into `rating_history` under that season id (D-12, BEFORE any mutation), then soft-resets active ratings — `new_rating = 1500 + (old_rating − 1500) × 0.5`, `rd = 350`, `volatility = 0.06`, `games_played = 0` (D-11).
