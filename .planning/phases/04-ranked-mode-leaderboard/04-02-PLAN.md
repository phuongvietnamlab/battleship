Slice 2 of the ranked vertical: make "ranked" a real, demoable game state end-to-end BEFORE the Phase-5 queue exists. A host toggles "Ranked" in the lobby (D-01), the server stamps `room.ranked`, and both the create and join paths reject guests with the named code `RANKED_REQUIRES_ACCOUNT` (D-02).
